🌳 Tree Commands

  • show TreeVisualizer.print_tree()
    Recursive DFS, draws connectors with color     DFS
  • search <query> TreeOperations.search()
    DFS traversal, matches key/value case-insensitive     DFS
  • get <path> TreeOperations.get_node()
    Dot-path walk: users.0.name, supports [n] indexing     Path Walk
  • info TreeOperations.get_stats()
    Recursive count of nodes + max depth     DFS

🕸️ Graph Commands (derived from tree)

  • → TreeGraphBuilder.build_from_tree(root) Walk tree → add nodes + parent→child edges
    Scan values for cross-ref keys: ref, link, depends_on, target…
    Add cross-reference edges via value matching / prefix ref:
③ Graph Structure — GraphModel
🕸️ GraphModel adj: dict[str, set[str]] · node_meta · alias_to_node

🕸️ Graph Operations

  • graph-info node_count() · edge_count() · has_cycle()
    Simple counts + cycle check
  • graph-bfs <start> GraphOperations.bfs_order()
    BFS using deque — returns ordered visit list     BFS
  • graph-path <start> <end> GraphOperations.shortest_path()
    BFS + parent-pointer backtrack to reconstruct path     BFS
  • graph-cycle GraphOperations.has_cycle()
    DFS with 3-color marking: 0=unvisited 1=visiting 2=done     DFS
